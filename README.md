# Projeto Store Manager

  - Nesse projeto fui responsável pela criação de uma API CRUD Restful em node utilizando a metodologia MSC para estruturação das camadas da api, fazendo consultas a um database em MySql, desenvolvendo uma cobertura de testes de 60%.

# Habilidades desenvolvidas

  - Docker compose/docker file;
  - Utilização da metodologia MSC;
  - Node.js;
  - Express;
  - MySql e suas queries;
  - Desenvolvimento de testes unitários para cada camada da api;
  - Documentação com o Swagger

# Para rodar o projeto

  - Git clone nesse repositório;
  - Npm start;
  - Inicie o container Docker;
  - Faça as requisições à api de acordo com a documentação;
